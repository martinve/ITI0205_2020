Fetch API Examples

The basic format of Fetch API request is:
  • The primary formas are json or text
  • The options is an object and may include { mode: 'no-cors' } to avoid CORS on client side or { method: 'POST'} to set request method type
See also examples and good explanation at: https://javascript.info/fetch

The server is running on port 8000 meaning the API endpoint for server is http://localhost:8000. In server we have installed cors module module to avoid cross-origin queries. Check the browser console and
